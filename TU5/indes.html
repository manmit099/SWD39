HTML Images, Lists & Tables

Concept explanations

HTML Images
Images are embedded in HTML using the <img> tag. It requires two main attributes: src (source path) and alt (alternative text). You can also provide width and height attributes to prevent layout shifts.
HTML Tables
Tables arrange data in rows and columns. Use <table> as the wrapper, <tr> for rows, <th> for headings, and <td> for data cells. Grouping is done with <thead>, <tbody>, and <tfoot>.
Rowspan & Colspan
These attributes merge table cells. colspan spans a cell across multiple columns (horizontally), while rowspan spans a cell across multiple rows (vertically).
HTML Lists
Lists structure itemized data. An Unordered List <ul> displays bullets, an Ordered List <ol> displays numbers/letters, and a Definition List <dl> displays terms and descriptions.

  </tbody>
</table>

[04]

Tips & Warnings

Always provide an alt attribute for images. It helps with SEO and displays text if the image link is broken.
Providing width or height to images helps the browser allocate space, preventing layout shifts on page load.
Semantically structure tables using <thead>, <tbody>, and <tfoot>. It helps with future CSS styling.
Use Replit for coding on your mobile if your computer is slow or unavailable!
Do not specify BOTH width and height manually unless you are sure of the aspect ratio. Setting both incorrectly will stretch and distort your image.

Step-by‑step walkthrough

01
Setup Environment
Create a new folder video5, open VS Code, create index.html, and type ! to generate Emmet boilerplate.
02
Add an Image
Use the <img> tag. Include src="image.png" to link the file and set an alt description. Set the width to scale it properly.
03
Create a Basic Table
Write <table>. Add a table row <tr> with header cells <th> (Name, Designation, Language). Add data rows with <td>.
04
Merge Table Cells
Add colspan="2" to a <td> to make it span horizontally across 2 columns. Try rowspan="2" to merge cells vertically.
05
Group Table Sections
Wrap the header row in <thead>, the main data rows in <tbody>, and add a footer row wrapped in <tfoot>.
06
Create Lists
Create bullet points using <ul> and list items <li>. Experiment with ordered numbering by switching to <ol> and modifying the type attribute (e.g., "A", "i", "1").

Quick summary

Images use the empty <img> tag, relying on src and alt attributes.
Setting just width or height on an image auto-adjusts the other dimension.
Tables are constructed row by row using the <tr> tag.
<th> makes table text bold and centered (headers), while <td> holds standard data.
colspan spans cells left-to-right; rowspan spans cells top-to-bottom.
<caption> can be used to set a title immediately inside the <table> tag.
Unordered lists <ul> generate bullet points for non-sequential items.
Ordered lists <ol> generate automatic numbers/letters based on sequence.
The type attribute changes bullet styles (circle, square) or numbering styles (A, i, 1).
Definition lists <dl> pair terms (<dt>) with descriptions (<dd>).
