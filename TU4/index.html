[01]

Concept explanations

Headings (h1 - h6)
HTML provides six levels of headings. <h1> is the main or top-level heading, while <h6> is the lowest. They are used to create a semantic hierarchy for your web page content.
Paragraphs
The <p> tag allows you to define regular text paragraphs. It automatically adds spacing before and after the text block, helping to organize blocks of information.
Anchor Tags & Links
The <a> tag creates a hyperlink to web pages, files, or locations within the same page. It uses the href attribute to specify the destination URL. A visited link automatically changes color.
Emmet & Lorem Ipsum
Emmet is a plugin in VS Code that allows fast typing. Using lorem followed by a number (like lorem23) generates dummy placeholder text useful for filling layouts.

[03]

Syntax explained

SYNTAXDESCRIPTION
<h1> ... </h1>Defines the most important heading on a page. Hierarchy goes up to h6.
<p> ... </p>Defines a paragraph. Adds spacing naturally around text blocks.
<a href="...">Anchor tag to define a hyperlink. The href attribute specifies the destination URL.
target="_blank"An attribute used inside anchor tags to open the linked document in a new tab/window.
lorem23Emmet abbreviation shortcut to quickly generate 23 words of random dummy text.
[04]

Tips & Warnings

Use Alt + Shift + Down Arrow in VS Code to quickly duplicate the current line downwards.
Use Alt + Click in VS Code to spawn multiple cursors, letting you type the same text in multiple places at once.
If text overflows the screen in VS Code, toggle "Word Wrap" (View > Word Wrap) to eliminate the horizontal scrollbar.
Wrapping elements in <p> tags simply to stack them (like we did for links here) is a "dirty way". We will learn proper HTML Lists later!
